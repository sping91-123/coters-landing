① 새로운 BingX 계정 생성하기
  1. 코털스 레퍼럴 링크로 접속
    👉 BingX 코털스 가입 링크 바로가기

    코털스 커뮤니티에서 제공하는 공식 레퍼럴 링크를 통해 BingX에 접속해 주세요.
  2. 레퍼럴 코드 확인 및 새 계정 생성
    ⚠️ 중요! 가입 페이지에서 레퍼럴 코드(초대인 코드)에 COTERS가 보이는지 반드시 확인하세요!

    ✅ 확인 사항:
    • 레퍼럴 코드/초대인 코드: COTERS
    • 새로운 이메일 주소로 계정 생성 (기존 계정과 다른 이메일)
    • 비밀번호 설정 및 기본 정보 입력
확인 완료!
새 계정이 생성되었습니다! 이제 기존 계정의 자산을 옮기는 단계로 진행하세요.
② 기존 BingX 계정의 자산 옮기기
  1. 기존 계정의 모든 자산을 다른 거래소로 이동
    🚨 매우 중요! 기존 BingX 계정에 있는 모든 자산을 다른 거래소에 잠시 옮겨두세요!

    📤 자산 이동 방법:
    1) 기존 BingX 계정에 로그인
    2) 현물 지갑에서 출금(Withdraw) 선택
    3) Gate.io, OKX, Bitget 등 다른 거래소로 전송
    4) 선물 포지션이 있다면 모두 청산
    5) USDT, BTC, 모든 코인을 완전히 출금
  2. 자산이 남아있지 않은지 꼼꼼히 확인
    ✅ 반드시 확인해야 할 항목:
    • 현물 지갑: 모든 코인 잔액 0원
    • 선물 계좌: 모든 포지션 청산 및 잔액 0원
    • 자금 계좌: 모든 자산 0원
    • 쿠폰/바우처: 사용 가능한 것들 모두 확인

    💡 팁: 소액(1~2달러 이하)은 남아도 괜찮지만, 가능한 한 모두 출금하는 것이 좋습니다.
주의사항
• 자산이 남아있으면 KYC 이전이 불가능할 수 있습니다
• 출금 수수료를 고려하여 출금하세요
• 출금 처리 시간은 보통 10~30분 정도 소요됩니다
③ 기존 계정의 KYC를 새 계정으로 이전하기
  1. 기존 BingX 계정에 로그인
    BingX 앱에서 기존 KYC가 인증되어 있는 계정으로 로그인해 주세요.
    (KYC 레벨 2 인증이 완료된 계정이어야 합니다)
  2. Verification 메뉴 접속
    1) 앱 좌측 상단의 동그라미 프로필 아이콘을 클릭
    2) "Verification" (또는 "신원 인증") 메뉴를 클릭
  3. Identity Transfer 시작
    1) "View Detail" (세부 정보 보기) 버튼 클릭
    2) 페이지 하단의 "Identity Transfer" (신원 이전) 버튼 클릭
  4. 본인 인증 진행
    1) 약관 동의 체크박스를 체크하고 "OK" 클릭
    2) 이메일 인증 코드 입력 (기존 계정 이메일로 전송됨)
    3) Google OTP 인증 코드 입력 (2단계 인증이 설정되어 있는 경우)
  5. 새 계정 정보 입력
    1) 새로 가입한 계정의 이메일 주소 또는 전화번호 입력
    2) 해당 이메일/전화번호로 전송된 인증 코드 입력
    3) "Submit" (제출) 클릭
  6. KYC 이전 완료 확인
    다음과 같은 메시지가 표시되면 KYC 이전 완료!

    ✅ "Identity transfer successful"
    또는
    ✅ "신원 이전이 완료되었습니다"

    이제 새 계정에서 기존 KYC 정보를 그대로 사용할 수 있습니다!
    기존 계정은 자동으로 탈퇴 처리됩니다.
  7. 자산 이동 및 심화반 인증
    1) 다른 거래소로 옮겨둔 자산을 새 BingX 계정으로 다시 입금
    2) 100달러 이상 예치 후 텔레그램으로 인증샷 전송:
    • UID 인증샷
    • 레퍼럴 확인 화면 (COTERS 표시)
    • 시드 인증샷 (100달러 이상)

    🎉 심화반 혜택을 받으실 수 있습니다!
KYC 이전 성공 팁 💡
✅ 반드시 확인하세요:
• 기존 계정의 모든 자산을 출금한 후 진행
• KYC 레벨 2 인증이 완료된 계정이어야 함
• 새 계정은 레퍼럴 코드 COTERS로 가입
• 이메일 인증과 OTP 인증 코드를 정확히 입력
• KYC 이전은 1회만 가능하므로 신중하게 진행
주의사항 ⚠️
• KYC 이전은 계정당 1회만 가능합니다
• 기존 계정에 자산이 남아있으면 이전이 안 될 수 있습니다
• KYC 이전 후 기존 계정은 자동으로 탈퇴됩니다
• 새 계정의 이메일/전화번호를 정확히 입력하세요
• 문제 발생 시 BingX 고객센터에 문의하세요
• KYC 이전이 불가능한 경우, 새 계정으로 다시 KYC 인증을 진행해야 합니다
