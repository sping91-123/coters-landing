## 🎯 프로젝트 개요

코털스는 암호화폐 거래소 수수료 페이백 정보와 전문적인 트레이딩 교육을 제공하는 커뮤니티입니다. 최대 50%의 수수료 페이백과 체계적인 심화반 교육을 통해 효율적인 암호화폐 투자를 지원합니다.

## 💡 사용 방법

### 개발 환경에서 실행
1. 프로젝트 폴더를 열기
2. `index.html` 파일을 브라우저에서 열기
3. 로컬 서버 사용 권장:
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js
   npx serve
   ```

1. **Publish 탭 사용** (권장)
